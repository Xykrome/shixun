mini_webserver V1.5

项目概述

基于 epoll 的配置驱动 HTTP 服务器，支持 JSON 配置文件加载、 三种认证方案（Basic / Session / Bearer Token）、路由分发、 静态文件服务和动态学生信息查询。

配置驱动 (V1.4)

启动时读取 server.json，校验 11 项配置字段，构建路由表。 修改端口/路由/日志无需重新编译，配置错误时拒绝启动并打印原因。

动态查询 (V1.3)

GET/POST 双模式学生信息查询，支持仅班级、仅关键词、班级+关键词 三种查询方式，跨班级搜索，URL 解码 + HTML 转义。

认证系统 (V1.5)

HTTP Basic 认证（401 挑战 + Base64 凭据校验）、Session 会话认证 （Cookie HttpOnly/SameSite + CSRF 防护）、Bearer Token 认证 （Token 签发/过期/吊销），三种方案按路由独立配置。
