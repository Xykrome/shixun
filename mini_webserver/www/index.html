W3D2 实训笔记

今天完成了静态文件服务器的构建。核心工作：

  • URL 路径映射：将请求路径安全映射到 www/ 目录
  • MIME 类型识别：根据文件扩展名返回正确的 Content-Type
  • 目录穿越防护：使用 realpath() 限定 document root
  • 分块文件发送：使用 8KB 固定缓冲区 + send_all()

技术栈

  • I/O 多路复用：epoll (LT + EPOLLIN)
  • HTTP 协议：HTTP/1.1 请求解析 + 响应构造
  • 文件服务：stat() + open() + read() 分块发送
  • 安全机制：路径规范化 + realpath() 边界校验
  • 日志系统：双文件日志（system.log + access.log）
