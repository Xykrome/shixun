mini_webserver V1.3

项目概述

基于 epoll 的轻量级 HTTP 服务器，支持静态文件服务、MIME 类型识别、 目录穿越防护和动态学生信息查询。

静态服务 (V1.2)

URL 路径安全映射到 www/ 目录，15 种 MIME 类型识别，realpath() 目录穿越防护，8KB 固定缓冲区分块发送。

动态查询 (V1.3)

GET/POST 双模式学生信息查询，支持仅班级、仅关键词、班级+关键词三种查询方式，跨班级搜索，URL 解码 + HTML 转义。

技术栈

epoll (LT + EPOLLIN) 单线程事件循环，HTTP/1.1 协议解析，双文件日志系统（system.log + access.log），统一 fd 清理。
