mini_webserver V1.3

项目概述

基于 epoll 的轻量级 HTTP 服务器，支持静态文件服务、MIME 类型识别、 目录穿越防护和动态学生信息查询。

静态服务 (V1.2)

  • URL 路径 → www/ 安全映射
  • 15 种 MIME 类型识别
  • realpath() 目录穿越防护
  • 8KB 分块 read() + send_all()

动态查询 (V1.3)

  • GET /search 搜索表单
  • POST /search 表单查询
  • URL 解码 + 参数校验
  • HTML 转义防 XSS 注入

技术栈

  • epoll (LT + EPOLLIN)
  • HTTP/1.1 协议解析
  • 双文件日志系统
  • 统一 fd 清理
