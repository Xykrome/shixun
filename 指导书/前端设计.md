# 一、项目整体目标

在 mini_webserver V1.2 基础上新增学生信息查询功能。

访问：

```
http://127.0.0.1:8080/search
```

显示一个查询页面。

支持：

- GET 查询
- POST 查询

数据来源：

```
data/2011.txt
data/2012.txt
...
```

文件内容格式：

```
20111010001 张三 男
20111010002 李四 男
20111010003 王五 女
```

查询逻辑：

用户输入：

```
班级：2011
关键词：男
```

服务器读取：

```
data/2011.txt
```

在每一行中查找关键字。

匹配到：

```
20111010001 张三 男
20111010002 李四 男
```

则显示在结果页。

# 三、GET 查询页面行为

访问：

```
GET /search
```

返回上述表单。

表单：

```
<form action="/search" method="GET">
```

提交后：

```
/search?class=2011&keyword=男
```

# 七、错误页面描述

## 班级格式错误

例如：

```
class=abc
```

显示：

```
400 Bad Request

班级格式错误
```

------

## 数据文件不存在

例如：

```
class=9999
```

文件：

```
data/9999.txt
```

不存在。

显示：

```
404 Not Found

班级数据不存在
```

------

## 请求方法错误

例如：

```
PUT /search
```

显示：

```
405 Method Not Allowed
```

响应头：

```
Allow: GET, POST
```
